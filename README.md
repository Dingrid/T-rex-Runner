# T-rex Game
Também conhecido como "jogo do dinossauro" ou "jogo do offline" essa é uma réplica simples feita em windows form utilizando C#.

## Como Jogar
- Tecle espaço para pular
- Tecle R para reiniciar o jogo (quando morrer)
